## 📚 Bienvenidos al Bootcamp

Este repositorio ha sido creado para acompañar el proceso formativo del **Bootcamp Full Stack Java Trainee 0052-3**, una iniciativa impulsada por **Talento Digital para Chile** y **SENCE**, orientada a desarrollar competencias técnicas altamente demandadas en la industria tecnológica actual.

Aquí encontrarás todo el material práctico, ejercicios guiados, desafíos, ejemplos de código y proyectos desarrollados durante el programa.

El objetivo es que cada participante pueda fortalecer sus habilidades como desarrollador Full Stack, comprendiendo tanto la lógica del backend como la construcción de interfaces y aplicaciones modernas.

---

## 🎯 ¿Qué aprenderemos?

Durante este bootcamp trabajaremos en áreas fundamentales del desarrollo de software:

### ☕ Backend
- Java
- Programación Orientada a Objetos (POO)
- Colecciones y Streams
- Manejo de Excepciones
- JDBC
- JPA
- Spring Framework
- Spring Boot
- APIs REST

### 🗄️ Bases de Datos
- SQL
- Modelamiento de Datos
- MySQL/ Postgres
- Consultas avanzadas
- Relaciones entre tablas

### 🌐 Frontend
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Consumo de APIs

### 🔧 Herramientas Profesionales
- Git
- GitHub
- Control de Versiones
- Buenas Prácticas de Desarrollo
